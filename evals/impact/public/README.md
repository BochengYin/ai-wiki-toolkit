# Public Materials

This directory holds externally readable summaries of the impact-eval work.

- `ai_wiki_impact_eval_pilot.md`: public pilot write-up with methods, results, limitations, and artifact links.
- `coding_agent_eval_harness.md`: Project A harness brief covering the task corpus, strategy matrix,
  metrics, reports, failure taxonomy, and path to a larger benchmark.
- `../reports/project_a_diagnostics_2026-06-04.md`: current verification pass and optimization
  backlog for the Project A harness.
- `agent_failure_postmortem_pack.md`: ten concise postmortems covering coding-agent and eval-harness
  failure modes found during the impact-eval work.
- `blinded_applies_when_route_replay.md`: route-quality ablation testing whether blind
  source-session-derived `applies_when` labels improve the 57-trace route replay.
- `route_precision_research_journey.md`: step-by-step route precision research report from broad
  labels and blinded metadata failures to mode-aware, workflow-aware router design.
