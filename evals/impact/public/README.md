# Public Materials

This directory holds externally readable summaries of the impact-eval work.

- `ai_wiki_impact_eval_pilot.md`: public pilot write-up with methods, results, limitations, and artifact links.
- `flask_swe_chain_memory_eval_report.md`: public case study on Flask SWE-Chain trial-error and recurring memory experiments.
- `flask_swe_chain_agent_skill_writeback_report.md`: public follow-up on runner-hook, conversation-fork, agent-skill writeback as the next product direction.
- `flask_swe_chain_dogfood_no_router_report.md`: public follow-up showing that dogfood without the router layer matched the strongest Flask SWE-Chain writeback results.
