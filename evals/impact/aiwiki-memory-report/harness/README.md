# Harness

This directory documents the SWE-Chain experiment harness used to produce the
AI Wiki memory report data.

The harness scripts live in external SWE-Chain experiment checkouts. This
directory records the public contract and selected representative snippets
without exposing local absolute paths or copying large run artifacts.

Files:

- `runner-entrypoints.md` - launcher and Python runner shape.
- `step-lifecycle.md` - per-migration execution flow.
- `artifact-contract.md` - what `chain.json`, `eval.json`, and logs mean.
- `reliability-model.md` - current guardrails and still-needed automation.
